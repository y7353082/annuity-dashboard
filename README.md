# 年金投资管理数据汇总

22家机构企业年金季度业绩数据录入、排名与导出工具。

## 在线访问

- **年金数据汇总**: https://你的用户名.github.io/你的仓库名/annuity-collector.html
- **PB-ROE策略工具**: https://你的用户名.github.io/你的仓库名/index.html

## 功能

- 22家机构季度数据录入与编辑
- 当年业绩排名表（单一/集合/其他计划）
- 滚动3年业绩排名（时间加权法）
- 全国企业年金基金投资收益率情况表（人社部格式）
- CSV/JSON 导入导出
- 批量导入历史数据

## 本地使用

直接浏览器打开 `annuity-collector.html` 即可，无需服务器。
